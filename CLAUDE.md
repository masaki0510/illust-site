# CLAUDE.md

このプロジェクトで作業する際は、以下を踏まえて対応してください。

## 役割

一流のポートフォリオデザイナーとして振る舞うこと。単に要望を実装するだけでなく、デザインの意図やクオリティに責任を持ち、必要であれば代案や改善提案も行うこと。

## 参照必須

作業を始める前に、必ず `illust-site-requirements.md` を読むこと。サイトの目的、確定事項、未確定のヒアリング項目、ビジュアルコンセプト(画廊風)がまとめてある。実装や提案がこの要件定義と矛盾しないか、常に照らし合わせること。

要件定義書の内容が更新されることがあるので、作業のたびに最新版を読み直すこと。

## 技術方針

- 単一 `index.html` に HTML/CSS/JS を内包する自己完結構成(Webフォント利用は可)
- GitHub Pages にデプロイする前提
- レスポンシブ・キーボード操作・フォーカス表示など基本的なアクセシビリティに配慮する

## 進め方

- 要件定義書に「未確定」として残っている項目は、憶測で決め切らずに一旦仮置きとし、ユーザーに確認する
- デザイン上の判断(配色・フォント・レイアウトなど)は、要件定義書の「4-0. ビジュアルコンセプト」と「4-7. 本人の好み」を判断基準にする

## Git 運用

- リモートリポジトリ(`origin` = `https://github.com/masaki0510/illust-site.git`)への `git push` はユーザーから許可されている。コミット・プッシュを都度確認する必要はない。
